# Introdução à Amazon Web Services: Computação em Nuvem

Para as pessoas sem formação técnica, pode ser difícil decifrar a Amazon Web Services (AWS). Este curso introdutório faz uma ponte entre não engenheiros e a nuvem, ajudando profissionais de funções não técnicas, como equipes financeiras, gerentes de projetos e profissionais de marketing, a aproveitar ao máximo a AWS. A instrutora Hiroko Nishimura apresenta um breve histórico da computação em nuvem, uma visão geral dos modelos de implantação de nuvem e um resumo dos princípios de projetos em nuvem.

Observação: este curso também está correlacionado com o primeiro domínio do exame AWS Certified Cloud Practitioner. Os quatro cursos da série Introdução à Amazon Web Services ajudam você a se preparar para o exame.

## Competências cobertas neste curso

- Computação em nuvem
- Amazon Web Services
